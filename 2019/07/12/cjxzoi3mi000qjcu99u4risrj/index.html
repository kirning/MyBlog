mysql8withNodeJs

由于mysql8.0的更新，mysql的默认认证方式从 mysql_native_password更换为caching_sha2_password所以导致 mysql 库用不了，报错

截止目前(2019-07-12)为止，官方仅仅提出了修复的PR,但是还没有合并到项目里面去。

目前能在网上找到的解决方案有两个：

  1. 设置数据库，使用原来的认证方式

  1. 使用 X DevAPI 库替换现在的库

    X DevAPI 是 mysql 官方推出的库。地址
