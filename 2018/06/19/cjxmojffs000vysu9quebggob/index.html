协议

2018年5月5日

机台发送给客户端

字段 类型 说明
clientId string 唯一识别码
col 2 is centered $12
col 3 is right-aligned $1

  1. 主题zbc/client/death 死亡主题:

    描述: 当客户端掉线的时候发送

    等级: >=1

    数据:

字段 类型 说明 例子
clientId string 唯一识别码 0201

  1. 主题 zbc/client/login 连接主题

    描述: 机台连接上来后发送

    等级: >=1

    数据:

字段 类型 说明 例子
clientId string 唯一识别码 0201

  1. 主题 zbc/client/circle 转圈主题

    描述: 每转一圈就发送一次数据到服务器

    等级: >=1

    数据:

字段 类型 说明 例子
clientId string 唯一识别码 0201
circle number 转数 20
totalCircle number 总转数 100
userId string 操作员ID sdfef
orderId string 当前布票ID dfdfdsdf
orderNum number 落布数 10

  1. 主题 zbc/client/ccd 扫描条形码数据

    描述: 机台扫描二维码后发送给服务器

    等级: >=1

    数据:

字段 类型 说明 例子
clientId string 唯一识别码 0201
code string 条形码数据 02215158

服务端发送给机台

  1. 主题 zbc/server/唯一识别码/changeUser 更换操作员

    描述: 机台更换操作员

    等级: >=1

    数据:

字段 类型 说明 例子
userName string 操作员姓名 张三
userId string 用户ID 26d51fd5f

  1. 主题 zbc/server/唯一识别码/login 机台历史数据

    描述: 机台登陆后,服务器发送给他历史数据,也就是断网前的数据

    等级: >=1

    数据:

字段 类型 说明 例子
clientNum string 机台号 001
circle number 转数 20
totalCircle number 总转数 100
userId string 操作员ID sdfef
userName string 操作员姓名 张三
orderId string 当前布票ID dfdfdsdf
orderNum number 落布数 10

  1. 主题 zbc/server/唯一识别码/ccd 布票数据

    描述: 机台CCD扫描后,服务端把布票数据发送给机台

    等级: >=1

    数据:

字段 类型 说明 例子
orderNum string 布票号 02123548

  1. 主题 zbc/server/唯一识别码/init 添加机台设备的时候发送

    描述: 添加设备的时候发一次给单片机,让单片机记录

    等级: >=1

    数据:

字段 类型 说明 例子
clientNum string 机台号 001

  1. 主题 zbc/server/唯一识别码/error 错误信息

    描述: 如果出现错误,服务端会把错误发送给机台

    等级: >=1

    数据:

字段 类型 说明 例子
msg string 错误信息 该员工已离职
